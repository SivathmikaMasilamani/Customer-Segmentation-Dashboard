# Customer Segmentation Dashboard

## Overview
This project analyzes customer demographics and purchasing behavior using Power BI.

## Features
- Total Customers KPI
- Average Income KPI
- Average Spending Score KPI
- Gender Distribution Analysis
- Age Group Analysis
- Interactive Filters and Slicers

## Tools Used
- Power BI Desktop
- CSV Dataset
- Data Visualization

## Key Insights
- High-income customers show higher spending potential.
- Customer behavior varies across age groups.
- Segmentation helps businesses improve targeted marketing strategies.
